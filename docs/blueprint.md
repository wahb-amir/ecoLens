# **App Name**: EcoLens

## Core Features:

- Image Upload: Allow users to upload images of waste for classification.
- Waste Classification: Send the uploaded image to the backend for classification using a CNN model. Uses an AI model to classify different types of waste (e.g., plastic, paper, organic).
- Classification Result Display: Display the classification result to the user with the type of waste and confidence level.
- Eco Tracker: Provide a personalized eco tracker to monitor the user's waste classification activities and their impact.
- Gamified Achievements: Reward users with badges for classifying different types and quantities of waste. Implements anti-cheat mechanisms to ensure fair gameplay.
- Global Leaderboard: Display a global leaderboard ranking users based on their waste classification scores and contributions to environmental conservation.
- AI-Powered Tip Tool: Provide users with eco-friendly tips and information related to waste reduction and recycling practices based on AI-analysis of users actions, and show why is incorreclty sorted waste affect the recycble rate every year and how does it affect the planet. Uses a tool to incorporate information about recycling rates.

## Style Guidelines:

- Primary color: Emerald green (#50C878) to reflect nature and environmental friendliness.
- Background color: Off-white (#F5F5DC) to create a clean and eco-friendly aesthetic.
- Accent color: Soft brown (#A0522D) to evoke feelings of earth and sustainability.
- Body and headline font: 'PT Sans' for a modern, clean, readable feel.
- Use nature-themed icons (leaves, recycle symbols, plants) to represent different categories and actions.
- Clean, card-based layout with a focus on readability and user-friendly navigation.
- Subtle animations for achievements and leaderboard updates to provide a sense of accomplishment.
